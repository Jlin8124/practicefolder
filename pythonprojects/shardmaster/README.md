# ShardMaster

`shardmaster.py` is a Python utility designed to manage, rebalance, and monitor database shards efficiently.

## Features 

* **Shard Allocation:** Automatically distributes data across available nodes.
* **Rebalancing:** Migrates chunks to ensure even load distribution.
* **Health Checks:** Monitors shard availability and latency.
* **Replica Management:** Handles primary/secondary failover logic.
